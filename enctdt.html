The Faculty of Computer Networks and Communications is in charge:

  • Engineer in Computer Networks and Data Communication

    • The Engineer in Computer Networks and Data Communication is a bachelor program in Can Tho University that was qualified with the international accreditation standard AUN-QA.

    The Engineer in CNC was qualified with the international accreditation standard AUN-QA.

    Computer Networks and Data Communication is a discipline that studies the principles of computer networks; the ways of designing and building a network system with different levels from local to global; the data exchange between network components; the development of network applications, web applications, and big data processing over the networks.

    The program aims to provide students with the theoretical, mathematical, and scientific foundations used in the computer networks and data communications industries. They are equipped with basic and in-depth knowledge to analyze, design, and propose solutions in network application development. Graduates are provided a solid background to be able to master the latest and advanced technologies such as cloud computing, grid computing, and Internet of things. Students are able to communicate in English for normal life and for their careers.

  • Engineer in Information Security

    • Engineer’s degree in Information Security provides students the capability of protecting digital information security in network systems; prevent and detect illegal access, modification, distribution, destruction, and attacks on an organization's information system; propose, monitor, assess, and verify security solutions; and handle problems related to security and system information safety.

    Students will be equipped with the basic knowledge and skills in data privacy/security/integrity, information systems, and computer networks; In-depth knowledge of cryptographic techniques, operating system security, security of Internet applications, hacking techniques, protection models and defense techniques against intrusion attacks; Ability to analyze problems, proficiently use advanced tools and solutions to ensure information security; Practice real-life scenarios for the prevention and detection of system attacks.

  • Cisco Certified Network Associate (CCNA)

    The CCNA program delves into the basic and important knowledge and skills of Computer Networking technology such as the OSI layers, the types of networks, package routing and switching with security, etc. This program helps students to be able to design, install, maintain, and repair network systems.

    Upon completion of the course, students will: receive a Certificate of Completion issued by Cisco Networking Academy; have the knowledge and skills to attend the international Cisco Network Specialist CCNA (200-301) competition through authorized Pearson VUE Test Centers.
